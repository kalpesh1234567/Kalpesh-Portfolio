## 🧑‍💻 How to Use

### 1. Clone the Repo

```bash
git clone https://github.com/kalpesh1234567/Kalpesh-Portfolio.git
cd Kalpesh-Portfolio
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the App

```bash
npm run dev
```

### 4. Build for Production

```bash
npm run build
```
